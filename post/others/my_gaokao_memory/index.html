高考回忆

简单的流水文，写于2025-06-07，今天起床时女朋友问我高考完多久了。我还仔细算了下，已经过去八年了。

如果你对一个人说八年，ta肯定会觉得很长，但如果让我回忆“高考”这件具体的事情，我会觉得我那年高考仿佛还在昨天。

tips: 都是流水式的回忆

考前#

高考考场离我们的教室隔了一两栋楼（本人也记不清了）。我们考生上考场前，各老师会穿上红衣服，手里拿着红色的旗子，在路的两边为我们加油打气。我和班主任对上了眼，他一直握住我的手“加油啊XXX”。

可惜后面让他“没有如愿”，哈哈哈。考得一般般。那会说不上很紧张，只是单纯地没有发挥出平时的水平。不过从后面的经历来看，高考对大部分人来说，是人生第一门大考了。

考中#

现在高考好像是考3天，还能自己选不同的科目组合去考。但是我那会还是最经典的“语、数、英必考，文综和理综选一个考”。一天考两门，总共就两天。

最后一门考英语，距离交卷还剩10分钟时。台上的监考老师提醒大家：“同学们，再过10分钟，你们的高中就结束了，最后在检查下自己的试卷。。。考完后和同学朋友再聚一聚，这将是你未来最无忧的时光”。那会我在台下听得津津有味，心里已经盘算这大学的生活了。

考后#

“叮叮叮~”，随着铃声响起，我的高中生涯也结束了。我和一个同班同学一起从考场走回教室，路上我们看着外面的天空，只是感慨”真tm热“。回到教室，大家都在玩闹，互相庆祝和扔书。那会我也想仍，但是怕第二年要复读，于是想着等分数出来再仍。班主任在讲台上仿佛想说啥，但是我们班太吵闹了，后来他只能叹气说算了。

后面他在班群上说了一句：”同学们，放学了…“

老实说，我还挺想听他念叨最后一下的，可惜了。

之后我爸来宿舍和我一起收拾东西，收拾完后我很平静地推开宿舍门打算走了。出门前我爸让我给宿舍拍张照片。

晚上一家人坐在一起吃饭，期间我一直吐槽这届试卷真难。现在看来，那时的我可能没有意识到，一个可能是最愉快的夏天就要来了。
