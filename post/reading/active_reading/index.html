为什么我们需要阅读#

很多人说这是一个好的时代（主要是中老年人），因为他们觉得这个时代获取信息相比他们那个年代来说太方便了。我们需要什么信息或资源，只需要打开手机查一下就可以了。特别是现在还有 ai 工具的加持，我们只需要输入一个关键词，就可以得到很多相关的信息。

但是他们错了，这个时代也是一个很“糟糕”的时代。的确获取信息和知识变得比以往任何时候都更易如反掌，但是你能保证你获取的信息都是高质量的吗？你能保证你获取到的知识都是对你有用，而不是浪费你的时间吗？如果你不能保证，那么这个时代对你来说就是一个“糟糕”的时代。

获取知识确实方便了，但是如今很多社交软件（比如某乎，某书，等等）为了吸引你的注意力，它们在产品设计或官方扶持的内容上，做了很多“别出心裁”的设计。这些设计的目的就是为了你在它们的 App 上多待一会儿。

关于它们的一些“设计”，包括不限于：

  • 当你搜索相关问题时，直接帮你整理好答案（甚至例举好 1，2，3）。让你无需思考即可得到答案。
  • 当你在看某个问题时，它会自动推荐相关问题，让你一直看下去。

基于这些“设计”，你会发现你在这些 APP 上花费的时间越来越多，你会沉迷在这种低质量的“满足感”中，而不是真正的知识。

诶？奇怪，我不是通过这些 APP 获取了很多知识吗？为什么你说我获取的知识是低质量的呢？

因为，它帮你整理好答案了呀，你不需要思考，你不需要去理解，你不需要去推理，你不需要去辩证，你不需要去总结，你不需要去应用。你只需要直接“拿来就用”。毕竟我们大脑是个耗能很高的器官，它想偷懒时就会本能地去偷懒。

知识的分类#

首先我们每个人都希望自己有出息，都会希望自己能成为梦想中的自己。在这个成为更好的自己的过程中，我们需要不断地学习，不断地积累知识。从事任何行业的人都是一样的（杠就是你对）。

而知识大致可以分为四类：

  1. 资讯，这个是最底层的，也是最容易获取的。比如你想知道今天的天气，你想知道今天的新闻，你想知道今天的股市行情等等。这些都是资讯，这些都是你可以在任何时候，任何地点，通过任何渠道获取到的。我们常说的信息爆炸中的信息指的就是这个。
  2. 知识，这个是比资讯高一层的。知识是对资讯的整理，是对资讯的归纳，是对资讯的总结。知识是有价值的，是有用的，是可以帮助你解决问题的。
  3. 理解，你通过各种渠道得到知识后，在这个基础上，加上你自己的归纳和总结，你对这个知识有了更深层次的理解。这个理解是你自己的，那么就可以说你真正掌握了这个知识。
  4. 娱乐，字面意思。比如网络小说，图个乐呵。

资讯是随处可见的，老实说，它最没有价值。知识是有价值的，比如关于一道菜的做法，它就算是知识。而理解是更进一步的东西，你脑子里知道了某个菜的做法，但是没法实际做出来，那这个知识就不属于你。只有你通过实践，通过自己的思考和总结，你才能真正掌握这个知识。这也是我们平时学习某个东西时，所要追求的境界。

当一个知识点，从普通“知识”变为“理解”之后，你的内心会真正的快乐，因为你会感觉自己变得更有价值了。但是获取知识的过程中，还有另外一种形式的快乐。那就是上文说的“娱乐”，你可以通过阅读爽文来获得，这种快乐是即时，马上生效的。

两种快乐有很大的区别，前者会让你由衷的感到快乐，它留在你体内的时间会更长。而后者，不是要完全禁止这种快乐，而是要克制。因为你长时间沉迷后者这种即时的快乐的话，你会感到空虚，会产生一种浪费时间的“罪恶感”。

所以我们平时在获取知识的过程中，可以适当的娱乐一下，但是不要沉迷。更应该了解当前自己需要的知识，并吸收成为自己的一部分。

回归主题，为啥我们需要阅读#

相信大家都听过这样一句话：

读书是性价比最高的投资

读书是性价比最高的投资，是因为读书是获取知识的最好途径。我们可以只花几十元或者自己找免费的电子书资源，就可以获得到一本书。通过书籍，我们相当于和作者进行了一次“对话”。

在一本不错的书中，作者会把他的感悟、心得等等都写在书中。通过阅读，我们可以汲取到作者的所思所想，了解另外一个人的学识或内心世界。

而且阅读注定是一件比较耗时间的事情。在一次高质量的阅读中，我们不可避免地需要思考与理解书中的内容。长久以往，这会锻炼我们的心性，使得其更有耐心。也会提升我们的学识（毕竟读了那么多本了，只要你不是囫囵吞枣）

所以大家看，读书是多么有意义的事情呀。但是它也很虐人，还是那句话我们的大脑不喜欢耗能高的活动，凡是需要思考的东西，都会被大脑本能的排斥。

关于上面的问题，我想说：

没办法了。

我的解决方案是自我PUA。就是反复告诫自己，这么做会对自己有收获，同时会想很多案例，长久以往，我真的对阅读或思考有兴趣了。获取是因为得到了延时满足吧。

不管各位如何克服上述的问题（如果你天生就很喜欢阅读，那恭喜你），我们都需要阅读。因为阅读是获取知识最好的途径，是提升自己的最好途径。

主动性阅读#

为什么同样的两个人，同时去读一本书，有些人读得更好呢？（理解更深）

答案是：主动性阅读。

阅读也是一次和作者的沟通。平时我们沟通是用语言，而阅读是用文字。我们每读一本书，就相当于和作者进行了一次对话。作者会抛出一系列的思想，而我们作为读者也是需要主动去“捕获”，不仅仅是被动的接受（作者说啥，读者就信啥）。

试想一下，平时我们和他人沟通时，如果他问一句，你也回答一句，不继续他的问题来扩展联想或反问，长久以往，对方会觉得你是个“呆子”。同样的，你也会觉得这种对话没有营养，读书也是如此的。

所以我们在阅读时，需要调动自己的“积极性”，去主动地思考与理解。

上面这句话是个漂亮话，具体怎么个主动法呢？下面慢慢来。

阅读时需要提问题#

阅读书时需要主动，这个主动简单说就是你需要不断地提问题并自己去寻找答案。

提问题的过程会督促你对作者的表达进行思考，找答案的过程相当于你对作者的表达进行一次辩证。这样你就会更深刻的理解书中的内容。哪怕一本书对你一无所获，这其中也锻炼了你的思考能力。

你在读书时可以提很多千奇百怪的问题，但是有4个基础的问题你需要知道：

  1. 一本书在说什么 (主题是什么)
  2. 一本书怎么说的 (怎么表达主题的)
  3. 说得有没有道理 (表达的内容是否有逻辑性)
  4. 我有关系吗 (对我有什么帮助)

做笔记#

主动阅读不仅仅是提问题与回答，同时还要有做笔记的习惯。

古人云：好记性不如烂笔头，这句话还是有道理的。做笔记主要是为了逼着我们做“输出”，把从书中获取到的知识进行归纳整理。

你想呀，当你读完书之后再逼着自己做笔记 => 动笔前脑中回想知识并组织语言 => 动笔时边写边调整 => 结束后在回顾一遍 => 这个过程中你会对书中的内容掌握更深。

做笔记的方式有很多，但是有三种笔记是我推荐的：

  1. 结构笔记，这种笔记是对书中的结构进行整理。比如书中的章节，每章节的内容是什么，每章节的内容之间有什么联系等等。
  2. 概念笔记，这种笔记是对书中的概念进行整理。比如书中的一些概念，这些概念是什么，这些概念之间有什么联系等等。
  3. 辩证笔记，对其他类似的书中的观点进行整理。比如你读了一本介绍社会经济学的书，你也可以多摘取其他有关社会经济学的书中的观点，进行整理，同时辩证地看不同书中的观点。

其实笔记和提问题是类似的，比如结构笔记本质上就是再问一遍“这本书在说什么”，概念笔记本质上就是再问一遍“这本书怎么说的”，辩证笔记本质上就是再问一遍“说得有没有道理”。

最后#

当生活折磨你时，书籍或许可以还安慰你。当生活善待你时，书籍或许可以让你更加快乐。

我还是初级读者，共勉啦。
